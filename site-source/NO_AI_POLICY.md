# No-AI Dependency Policy

AI tools must not be used to generate complete answers, full code, checkpoint submissions, debugging reports, or architecture explanations.

Allowed:

- official documentation
- Stack Overflow
- GitHub issues
- web search
- YouTube explanation videos
- mentor review after you show evidence

Restricted:

- asking AI to build the project
- asking AI to fix the entire project
- asking AI to write the checkpoint explanation
- submitting AI-written text without understanding

The expected learning loop is:

```text
observe → search → read → try → fail → inspect → fix → explain
```

If you cannot explain your work during review, the checkpoint is incomplete.
